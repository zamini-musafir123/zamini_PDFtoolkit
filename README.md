## ✨ What It Does

Zamini PDF Toolkit has **6 fully featured tabs**, each solving a real problem:

### 📄 Merge
Combine multiple files — PDFs, Word docs, images, Excel sheets — into one clean PDF.  
- Add Files or entire Folders in one click  
- Drag & drop to reorder  
- Extra Page for double-sided printing  
- Splitter pages between sections  
- Export as Single PDF or Separate PDFs

### ✂️ Split
Extract exactly the pages you need from any PDF.  
- Split by Page Range, Keyword, or Sequence Step  
- Keyword mode searches inside page content  
- Interleave: insert a logo or back-page behind every certificate automatically  
- Mixed Rules: different split settings per file  
- Export as Single or Separate PDFs

### 🖊️ Stamp
Add professional stamps to every page of every PDF in one click.  
- Text mode with dynamic tokens: `{page}`, `{total}`, `{date}`  
- Image mode: stamp your logo or signature  
- 9-position placement grid  
- Opacity control for watermarks  
- Batch stamp multiple PDFs at once

### 📷 Album
Turn a folder of images into a professionally formatted PDF album.  
- Auto-captions from filenames (no typing needed)  
- Global caption settings: font, size, color, alignment — change once, updates everywhere  
- Layout selector: 1, 2, 4 images per page  
- Export as Single or Separate PDFs

### 🗜️ Compress
Reduce PDF file size without touching text quality.  
- 3 quality levels: ⚡ Smallest (72 DPI) · ⚖️ Balanced (96 DPI) · 💎 Sharpest (120 DPI)  
- Compress preview before saving  
- Batch compress multiple PDFs  
- Text stays sharp — only images are compressed

### 📐 Layout
Resize, rearrange, and reformat PDF pages for print.  
- N-Up Grid mode: 2-up, 4-up, up to 9-up (print 4 slides on one page)  
- Stretch or Fit scaling  
- Portrait / Landscape output  
- RTL mode for Arabic and Hebrew documents  
- Margin control with link/unlink toggle  
- Guided Wizard for first-time users
